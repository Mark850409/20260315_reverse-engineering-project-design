# 常見架構決策模板

## 資料庫選型 ADR

**觸發條件**：從 ORM 設定、連線字串、Migration 檔案識別

```
ADR-XXX：資料庫選型
背景：需要持久化儲存，有 [關聯資料/高寫入/全文搜尋] 需求
選項：PostgreSQL / MySQL / MongoDB / Redis
採用：[識別到的資料庫]
推測原因：[從 Schema 特性、索引設計、查詢模式推導]
```

## 認證機制 ADR

**觸發條件**：從 middleware、token 處理、session 設定識別

```
ADR-XXX：認證機制選型
背景：需要識別使用者身份，考量 [無狀態/跨服務/第三方整合]
選項：JWT / Session Cookie / OAuth2 / API Key
採用：[識別到的機制]
推測原因：[從程式碼模式推導]
```

## 前端架構 ADR

**觸發條件**：從 package.json、build 設定、元件結構識別

```
ADR-XXX：前端框架選型
背景：需要構建 [SPA/SSR/靜態] 介面
選項：React / Vue / Angular / Next.js / Nuxt
採用：[識別到的框架]
推測原因：[從依賴、慣例、結構推導]
```
